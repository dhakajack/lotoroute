## Geographic Metadata

The location-aware difficulty model uses a hand-curated metadata file containing coordinates, population estimates, graph neighbors, and practical ferry/tunnel adjustments.

- French department public reference data: https://geo.api.gouv.fr/
- Country capitals, population estimates, and country-neighbor relationships are compiled from public reference sources.
- Ferry/tunnel links and scoring adjustments are game-model assumptions documented in `README.md`.
